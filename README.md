#  React Counter App

A simple React application that allows users to *increment, **decrement, and **reset* a counter. It demonstrates the use of React hooks (useState and useEffect) and basic CSS styling.

---

##  Features
- Increment and decrement buttons  
- Reset button to set the counter back to zero  
- Smooth button hover effects  
- Responsive and minimal design  
- Background image support  

---

##  Technologies Used
- *React* – Frontend library  
- *JavaScript (ES6+)*  
- *CSS3* – Styling and layout  
- *Vite* – Fast React setup and local development
